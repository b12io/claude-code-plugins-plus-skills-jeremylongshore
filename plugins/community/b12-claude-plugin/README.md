# B12 Website Generator

B12 Website Generator plugin allows you to create a professional, engaging, and user-friendly website in seconds using AI. To create a website, you need to provide a name for your project/business, along with a description of the project/business _(goals, structure, etc.)_.

## What It Does

Claude automatically uses this skill for generating website link based on the provided project/business name and description.

## Installation

### Claude.ai

1. Download the [website-generator.zip](website-generator.zip) file
2. Go to **claude.ai** > **Settings** > **Skills**
3. Upload the zip file

### Claude Code

```bash
claude --plugin-dir /path/to/b12-claude-plugin
```

### Claude Cowork

Install through the plugin manager in Claude Desktop.

## Usage

```
“Create a website for consulting firm”

“Create a website for a software development agency”

“Create a personal website”

“Create a website for a cat café”
```

Claude will automatically create link to production-ready website.
